# SODAARS Layout Guidelines

## 1. Grid & Workspace Base
- **Standard Widths**: Maximum viewport usage for tables with a responsive fluid layout.
- **Side Panel Sidebar**: Left-docked navigation bar.
- **Top Header panel**: Page navigation breadcrumbs, quick global search bar, user notifications badge, and profile menu.

## 2. Responsive Rules
- **Desktop (1280px+)**: Multi-column grids and dense datagrids with action columns.
- **Tablet (768px - 1024px)**: Collapsed sidebar (hamburger menu), responsive scrollable data tables.
- **Mobile (767px or less)**: Grid layout cards instead of tables, full-width actions, and sticky drawer action sheets.
