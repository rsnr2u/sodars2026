# Asset Folder: UI Mockups

> **This document represents the finalized Version 1 architecture. Any new feature outside Version 1 must be documented under `12-future-roadmap.md` before implementation.**

## Purpose
The purpose of this folder is to store User Interface (UI) and User Experience (UX) wireframes, layout plans, and mockup images.

## Scope
This folder contains design mockups for the Head Office admin console, Branch manager views, Provider portal, public Marketplace site, and the React Native mobile app screens.

## Future Contents
In future development phases, this folder will house:
* UI wireframes for POS-style Campaign Booking screens.
* Public Google Maps search interface mockups.
* Customer cart layout and mobile screen captures.
* Provider dashboard mockups showing occupancy calendars.
