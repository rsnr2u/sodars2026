# Asset Folder: Logos

> **This document represents the finalized Version 1 architecture. Any new feature outside Version 1 must be documented under `12-future-roadmap.md` before implementation.**

## Purpose
The purpose of this folder is to store official branding graphics and logotypes of SODARS.

## Scope
This folder serves as the central directory for logos in various formats (PNG, SVG, JPG) and colors (light, dark, monochrome).

## Future Contents
In future development phases, this folder will house:
* High-resolution primary logotypes for dark and light background states.
* Horizontal and stacked logo orientations.
* App launcher icon logos for mobile stores.
* Clean vector SVGs for portal header integrations.
