# Asset Folder: Icons

> **This document represents the finalized Version 1 architecture. Any new feature outside Version 1 must be documented under `12-future-roadmap.md` before implementation.**

## Purpose
The purpose of this folder is to store interface icons, buttons, markers, and favicons.

## Scope
This folder covers SVGs and mobile asset files used inside the custom vanilla CSS components and the mobile application wrapper.

## Future Contents
In future development phases, this folder will house:
* Google Maps pin custom icon vectors (e.g., active screens vs inactive pins).
* Portal navigation sidebar icons.
* Site favicon variations (16x16, 32x32, 180x180 Apple Touch icons).
* UI status notification icons (e.g., success, error alert boxes).
